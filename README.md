## 📜 About

Lawful AI bridges the gap between AI engineering and legal compliance — the intersection most engineers ignore and most lawyers don't understand.

Every morning at 07:00 CET, automated agents scan GitHub releases, arXiv, regulatory feeds, and community repos. The results are reviewed, curated, and published as a daily intelligence brief.

Built with [OpenClaw](https://github.com/openclaw/openclaw) cron jobs, GitHub Actions, and Astro.
