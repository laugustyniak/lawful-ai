# Lawful AI — Repository Guidelines

- Content lives in `briefs/YYYY/MM/DD.md` — one file per day
- Site source in `site/` (Astro + Tailwind)
- Scripts in `scripts/` — feed generation and README updates
- Template in `templates/daily-brief.md`
- All briefs must have valid YAML frontmatter (date, title, tags)
- Tone: witty, opinionated, emoji-friendly. Not corporate.
- The Legal × AI Watch section is the differentiator — never skip it
- Do not edit feed.json manually — it's auto-generated
- README stats between markers are auto-updated by CI
