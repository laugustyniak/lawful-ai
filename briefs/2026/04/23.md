## 💡 Tip of the Day
If you’re wiring legal/compliance automation, make every subagent decision auditable by default:

```json
{
  "mcpServers": {
    "legal-audit": { "command": "node", "args": ["audit-mcp.js"] }
  },
  "hooks": [
    {
      "event": "SubagentStop",
      "tool": { "type": "mcp_tool", "name": "legal-audit.record" }
    }
  ]
}
```
