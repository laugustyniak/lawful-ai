## 💡 Tip of the Day

Treat trust as code, not vibes.

```toml
# .codex/config.toml
[project]
trusted = false

[security]
requireTrustForHooks = true
requireTrustForExecPolicy = true
```

Why this slaps: it blocks “surprise automation” until the project is explicitly trusted. Safer defaults, calmer mornings ☕
